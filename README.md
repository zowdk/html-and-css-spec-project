## User Stories

The wireframe should contain at least: 

    - Main Heading (h1)
    - Subheading (h2)
    - Main Image (img)
    - Text paragraphs (p)
    - Links (underlined blue text) (a)
    - Lists (ul, ol, li)
    - Sections (div)
    - Social media icons (identify what tag a social media icon is)
